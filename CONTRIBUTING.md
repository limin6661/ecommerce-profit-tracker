# Contributing to Ecommerce Profit Tracker PRO

Thanks for your interest! This repository hosts the marketing site, documentation, and the free profit calculator for **Ecommerce Profit Tracker PRO**.

## Ways to contribute

- 🐛 **Report a bug** in the [free calculator](https://limin6661.github.io/ecommerce-profit-tracker/calculator.html) — open an [Issue](../../issues)
- 💡 **Suggest a feature** — e.g. a new fee preset (TikTok Shop, WooCommerce), a new formula, a localization
- 🌍 **Translate** the calculator or guides into another language
- 📝 **Improve the seller guides** in `/guides/` — fix an error, add an example, clarify a calculation
- 🎨 **Improve the landing page** UI/UX or accessibility

## How to submit a change

1. Fork the repo
2. Create a branch: `git checkout -b fix/your-change`
3. Make your change
4. Commit: `git commit -m "fix: describe the change"`
5. Open a Pull Request against `main`

For calculator math changes, please include the worked example showing the expected result.

## Code style

- The landing page and calculator are plain HTML/CSS/JS — no build step. Keep it that way (instant load, no dependencies).
- Keep the free calculator dependency-free and client-side only.

## Issues with the paid product

The paid spreadsheet itself isn't open source, but if you find a formula error after purchasing, please [open an issue](../../issues) with the scenario and expected result — it'll get fixed and you'll get the updated file.

## Code of Conduct

Be kind. Seller communities thrive on generosity. Spam, self-promotion of unrelated products, and harassment will not be tolerated.

Thanks for making this better for fellow sellers! 🛒
